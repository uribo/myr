## 変数への代入

Rにおいて、変数への代入は「**->**」オペレータで行います。（=, <-）も可能ですが推奨されません。


```r
a <- c(1, 2, 3)
a
[1] 1 2 3
```

aというオブジェクトに数値ベクトル1, 2, 3を代入しました。

代入するのは数値でも文字でも構いません。

```r
msg <- c("Hello", "Taro")
msg
[1] "Hello" "Taro"
```

複数のオブジェクトに対し、同一の値を代入する場合、「->」オペレータで繋ぐことで可能になります。

```r
df1 <- df2 <- df3 <- c("test")
df1
[1] "test"
df2
[1] "test"
df3
[1] "test"
```
